## 一句話總結這個專案

> 在無人機上掛**熱像儀**,用 **Kneron KL730 這顆低功耗 AI 晶片**,跑 **YOLOX 這個物件偵測模型**,**機上即時**辨識畫面裡的**無人機 / 人 / 車**。
> 目前狀態:軟體鏈全打通(資料→訓練→量化→KL730 可執行檔),量化最大風險已解,正式訓練待上雲端 GPU。

## 學習路徑建議

- **只想懂大概** → 讀 L1 + L2,看完就懂「這在幹嘛、難在哪」。
- **要能跟工程師對話** → 加讀 L3,你會知道每個環節的名字與順序。
- **要自己動手 / 接手** → 加讀 L4 + L5 + 根目錄 `README.md`(操作手冊)。
- **隨時卡在某個詞** → 翻 `glossary.md`。
